# 🧙‍♂️ Wizard Rooms Game

Wizard Rooms is a strategic, grid-based dungeon crawler game played directly in your browser. Navigate through mysterious rooms, defeat demons, and manage your resources to survive the magic!


## ✨ Features

- **Procedural Room Generation:** Every run is unique based on a seed.
- **Strategic Combat:** Bump into enemies to attack while managing your health.
- **Resource Management:** Collect 💎 Diamonds, ❤️‍🩹 Healing items, and ⚡ Power-ups.
- **Progressive Difficulty:** 10 rooms per map with increasing challenges.
- **Modern UI:** Sleek dark mode with glassmorphism effects.

## 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YourUsername/WizardRoomsGame.git
   ```
2. **Open the game:**
   Simply open `index.html` in any modern web browser.
3. **Development Mode:**
   Use VS Code's **Live Server** extension for the best experience.

## 🎮 How to Play

- **Arrow Keys:** Move your wizard.
- **Bump Enemies (😈):** Attack enemies by walking into them.
- **F Key:** Use ⚡ Power (if ready) to freeze enemies.
- **R Key:** Restart the current run.
- **Objective:** Clear all enemies in a room to unlock the doors and advance to the next level.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Advanced layouts (Grid/Flexbox) and glassmorphism.
- **JavaScript (Vanilla):** Game logic, state management, and procedural generation.
